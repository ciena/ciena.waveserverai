Examples
--------

.. code-block:: yaml

    # Using merged

    - name: Disable ptp
      ciena.waveserver5.waveserver5_ptps:
        config:
          - ptp_id: 5-1
            state:
              admin_state: disabled
        state: merged
    - name: Enable ptp
      ciena.waveserver5.waveserver5_ptps:
        config:
          - ptp_id: 5-1
            state:
              admin_state: enabled
        state: merged
